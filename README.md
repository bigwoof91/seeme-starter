## For Local Testing and Development
in '/server' run:
```npm install```

in '/client' run:
```npm install```

make sure you're running mongoDB ```mongod``` in Bash or Terminal

in '/client/ run (install nodemon globally):
```nodemon ./node_modules/webpack-dev-server/bin/webpack-dev-server.js```

in '/server' run:
```nodemon index.js```

Good to go!
